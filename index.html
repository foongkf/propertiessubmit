房产智能同步系统

填写房地产资料 → 一键同步至 Google Sheets · 所有字段均可选留空

支持长串字母、符号、数字，留空亦可

📌 第一步：准备 Google Sheet 并部署后端脚本

  1. 新建一个 Google Sheets 表格（例如命名“房地产数据库”）。
  2. 点击菜单 扩展程序 → Apps 脚本，删除默认代码，完整复制下方提供的代码并粘贴保存。
  3. 修改脚本中的 表头字段（如有需要） 我们已经预设好中文字段，与表单完全匹配。
  4. 点击编辑器左上角 部署 → 新建部署 → 类型选择 Web 应用。
    - 执行身份：我；
    - 访问权限：任何人（甚至匿名）；
    - 点击“部署”，授权后得到 部署网址 (类似 https://script.google.com/macros/s/.../exec)。
  5. 复制该网址，稍后填入下方HTML代码中的变量 APPS_SCRIPT_URL。

📌 第二步：Apps Script 完整代码 (直接复制使用)

// ========== 请粘贴到 Google Apps 脚本编辑器 ==========
function doPost(e) {
  try {
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    const data = JSON.parse(e.postData.contents);
    
    // 定义表头（中文，与前端字段对应）
    const headers = [
      "时间戳", "标题(Title)", "地区(Area)", "土地面积(LandArea)", 
      "建筑面积(BuiltUp)", "卧室(Bedroom)", "浴室(Bathroom)", "状况(Condition)",
      "装修(Renovate)", "家私(Furnish)", "车位(Parking)", "产权(Tenure)",
      "交易类型(Type)", "价格(Price)", "独家(Exclusive)", "图片(Images)"
    ];
    
    // 检查首行，若无表头则写入
    const headerRow = sheet.getRange(1, 1, 1, headers.length).getValues()[0];
    if (headerRow.every(cell => cell === "" || cell === undefined)) {
      sheet.getRange(1, 1, 1, headers.length).setValues([headers]);
    }
    
    // 构建行数据（顺序严格对应 headers）
    const row = [
      new Date().toLocaleString("zh-CN", {timeZone: "Asia/Shanghai"}),
      data.title || "",
      data.area || "",
      data.landArea || "",
      data.builtUp || "",
      data.bedroom || "",
      data.bathroom || "",
      data.condition || "",
      data.renovate || "",
      data.furnish || "",
      data.parking || "",
      data.tenure || "",
      data.type || "",
      data.price || "",
      data.exclusive || "",
      data.images || ""
    ];
    
    sheet.appendRow(row);
    
    return ContentService.createTextOutput(JSON.stringify({
      result: "success",
      message: "房产数据已同步到 Google Sheet"
    })).setMimeType(ContentService.MimeType.JSON);
    
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({
      result: "error",
      error: err.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}

📌 第四步：将整个网页部署到 GitHub Pages (完全免费)
1. 创建一个新的 GitHub 仓库，例如 property-sync-system。
2. 将本 HTML 代码保存为 index.html (确保文件名准确)。
3. 在仓库中上传 index.html，然后进入 Settings → Pages → Branch 选择 main ，保存。
4. 几分钟后即可通过 https://你的用户名.github.io/仓库名/ 访问您的智能表单系统。
